<h1 align="center">Web Video Editor 在线视频编辑器</h1>

## 依赖
* Fabric.js  - 视频元素拖动定位
* React.js - 前端交互框架
* FFmpeg.wasm - 音视频处理
* Ant design - UI 框架
* Flexlayout-react - UI架子
